# Manual do Usuário

## O que é esta carteira?

InfiniteRicks Electrum Wallet é uma carteira **leve** para a moeda InfiniteRicks (RICK). Ela não baixa a blockchain inteira — sincroniza apenas os dados necessários via servidor SPV.

## Criar carteira

1. Toque em **Criar Carteira**
2. Anote as **12 palavras** exibidas em papel (nunca digitalize)
3. Toque **Copiar** se necessário, depois **Continuar**
4. Informe 3 palavras aleatórias para verificação
5. Defina uma **senha** forte

## Importar carteira

1. Toque em **Importar Carteira**
2. Digite as 12 palavras separadas por espaço
3. Defina senha e confirme

## Tela principal

- **Saldo** em RICK
- **Deposit Coins** — QR code e endereço para receber
- **Send Coins** — enviar RICK
- **Transactions** — histórico completo
- Status da rede: 🟢 conectado / 🔴 desconectado

## Enviar moedas

1. Informe o endereço destino
2. Digite o valor em RICK
3. Escolha taxa Normal ou Fast
4. Confirme o envio

## Backup

Em **Configurações → Backup**, a seed é copiada. Guarde offline em local seguro.

## Segurança

- Nunca compartilhe sua seed ou senha
- Chaves privadas permanecem no dispositivo
- A carteira usa criptografia AES-256-GCM
